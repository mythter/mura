# Stash

### Save changes

new

```shell
git stash
```

```shell
git stash push
```

```shell
git stash push -m "message"
```

old

```shell
git stash save "message"
```

### View stash content

```shell
git stash list
```

### Get content from stash (and delete from there)

```shell
git stash pop
```

### Get content from stash (and leave it there)

```shell
git stash apply
```

### Delete stash contents

```shell
git stash drop
```

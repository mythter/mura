Ветки

СОЗДАНИЕ

Создать ветку

git branch <branch-name>

Создать ветку и переключиться на неё

обычная команда

git checkout -b <branch-name>

новая команда

git switch -c <branch-name>

Если есть удалённая ветка, но вам нужно создать локальную и привязаться к удаленной

git branch --track <local-branch> origin/<remote-branch>

git branch <local-branch> origin/<remote-branch>

git checkout -b <local-branch> origin/<remote-branch>

Если локальная ветка с таким именем уже существует, то она будет отслеживать удалённую

git branch -f <local-branch> origin/<remote-branch>

Если есть локальная ветка и вам нужно создать удалённую с отслеживанием

git push -u origin <branch-name>

Если есть как локальная, так и удалённая ветки и вам необходимо связать их

git branch -u origin/<remote-branch-name> <local-branch-name>

Если вам нужно удалить отслеживание

git branch --unset-upstream <branch-name>

ПЕРЕМЕЩЕНИЕ

Переместить указатель ветки на другой коммит (destination-commit может быть названием ветки)

git reset --hard <destination-commit>

git branch –f <branch-name> <destination-commit>

УДАЛЕНИЕ

УДАЛИТЬ удалённую ветку (remote-name обычно origin)

git push -d <remote-name> <branch-name>

Не забудьте выполнить git fetch --all --prune или git fetch --all -p на других машинах после удаления удалённой ветки на сервере. После удаления локальной ветки с помощью git branch -d и удаления удалённой ветки с помощью git push -d origin на других машинах могут остаться "устаревшие отслеживаемые ветки" (чтобы увидеть их, выполните git branch -a). Чтобы избавиться от них, выполните git fetch --all --prune

УДАЛИТЬ локальную ветку

git branch -d <branch-name>

Опция -D это алиас для --delete --force, который удаляет ветку «независимо от её объединенного статуса (merged status)»

git branch -D <branch-name>

УДАЛИТЬ локальную удалённую ветку

git branch -dr <remote>/<branch>

длинная команда

git branch --delete --remotes <remote>/<branch>

ПЕРЕИМЕНОВАНИЕ

Переименовать текущую ветку

git branch -m <branch_name>

Переименовать другую ветки

git branch -m  <oldbranch_name> <new_branch_name>

ПРОСМОТР

Просмотреть объединённые ветки

git branch --merged

Просмотреть объединённые удалённые ветки

git branch -r --merged

Просмотреть не объединённые ветки

git branch --no-merged

УДАЛЕНИЕ УСТАРЕВШИХ ВЕТОК

git remote prune origin

короткая команда

git fetch -p

длинная команда

git fetch --prune
